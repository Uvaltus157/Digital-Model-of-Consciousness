Project goal

The project builds a digital model of consciousness as an embodied system with sensors, internal object imagery, a world model, memory, self core, motivation, metacognition, inner speech, debug visualization, and counterfactual imagination.

The central principle is a closed loop: sensing → internal objects → action → consequences → memory → self → broadcast → planning. No single module is “conscious” by itself; the conscious state is the coordinated runtime pattern.

Runner as the project skeleton

runner.py
Root launcher. Preserves the normal command python runner.py --config-path config --config-name runner.
↓
runner_entry.py
Thin Hydra entrypoint: resolved config → typed config object → system factory.
↓
runner_system_factory.py
Builds the system, applies runtime state, startup state, services and mode.
↓
UnifiedSystemV510
Compatibility wrapper assembled from mixins. Its init lives in runner_unified_init.py; its loop lives in runner_loop.py.
