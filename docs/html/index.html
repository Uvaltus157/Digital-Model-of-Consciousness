Digital Model of Consciousness

Архитектурная документация проекта

Проект строит цифровую модель сознания как систему из сенсоров, внутреннего объектного воображения, мировой модели, памяти, самости, мотивации, метакогниции, внутренней речи и отладочных визуализаторов.

Главная идея

Система должна не просто предсказывать кадры, а строить внутреннюю сцену: выделять устойчивые object slots, хранить латентные объекты, связывать их с действием и телом, превращать изменения в события, сохранять эпизоды и использовать их для replay, сна и планирования.

Верхний уровень проекта разделён на 15 смысловых модулей M1–M15. M1 создаёт внутренние объектные образы, M5 держит мировую модель и workspace, M9–M12 формируют ядро субъективности, а M13–M15 добавляют автобиографию, семантику и воображаемое планирование.

Текущий launch path

runner.py
Корневой compatibility launcher. Сохраняет привычную команду запуска.
↓
runner_entry.py
Тонкий Hydra entrypoint: печатает resolved config, строит config object, вызывает system factory.
↓
runner_system_factory.py
Создаёт UnifiedSystemV510, применяет runtime state, startup state, services и mode.
↓
runner.py / UnifiedSystemV510
Тонкий compatibility wrapper. Класс собран из mixin-ов, а основные hooks вынесены в helper-файлы.
↓
runner_unified_init.py
Фактическая инициализация системы: модель, object imagery, self core, replay, world, dynamic rig, visualizers.
↓
runner_loop.py
Life loop, train thread, MuJoCo viewer sync, shutdown, checkpoint save, world close.

15 модулей архитектуры

M1 Object ImageryСенсоры → fusion → object slots → latent object → 2D/3D/4D reconstruction.
M2 Event Dream ReplayИзменения slot-ов → события → токены → предложения → эпизоды → replay.
M3 Self Action CausalityДействия, action heads, ручной override, причинность действия.
M4 Long Dynamic MemoryДолгая динамическая память объектов и identity passport.
M5 World Model WorkspaceDreamer/RSSM, attention, workspace, novelty, planner.
M6 Learning SleepTraining loop, sleep gates, consolidation, replay training.
M7 Inner SpeechВнутренний текст, thought decoder, subjective stream.
M8 Debug Visual ControlModule Debug UI, IPC status, visual diagnostics.
M9 Self CoreТело, agency, ownership, egocentric frame, self-state.
M10 Conscious BroadcastГлобальный broadcast: кандидаты, gate, selected packet.
M11 MotivationCuriosity, safety, mastery, valence, arousal, urge.
M12 MetacognitionConfidence monitors, uncertainty, trust gates.
M13 Autobiographical MemorySelf episodes, timeline, discovery/mistake tags.
M14 Semantic GroundingGrounding слов в object/action/sensation/self.
M15 Imagination PlanningCounterfactual rollout, future scenarios, plan evaluation.
